# 个人常用termux命令
##  拉取所有的更新
```
cd ~/my/blogRoot/ && 
git pull origin src && 
cd ~/blog/blogRoot/ && 
git pull origin src && 
cd ~/blog/blog/ && 
git pull origin master && 
cd ~/my/blog && 
git pull origin master && 
cd ~/blog/JavaReadingNotes/ && 
git pull origin master && 
cd ~/my/JavaReadingNotes/ && 
git pull origin master && 
cd ~/blog/exam/ && 
git pull origin master && 
cd ~/my/exam/ && 
git pull origin master

```
