# 个人常用termux命令
##  拉取所有的更新
```
# 更新主站
cd ~/my/blogRoot/
git pull origin src
cd ~/blog/blogRoot/
git pull origin src

# 更新博客
cd ~/blog/blog/
git pull origin master
cd ~/my/blog
git pull origin master

# 更新读书笔记java
cd ~/blog/JavaReadingNotes/
git pull origin master
cd ~/my/JavaReadingNotes/
git pull origin master

# 更新面试题
cd ~/blog/exam/
git pull origin master

cd ~/my/exam/
git pull origin master

```
